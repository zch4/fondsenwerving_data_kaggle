# Fondsenwervingsdata - hobbyproject 

## Over dit project
Dit project gebruikt een mock-dataset van Kaggle over fondsenwerving en donateursdata. Het doel is om de donor- en transactiedata in te lezen, controleren, verrijken en om te zetten naar rapportageklare bestanden.

Wat dit project laat zien:
- CSV-bronbestanden inladen en structureren
- transacties en donateursdata controleren op datakwaliteit
- tabellen verrijken via joins
- rapportageklare exports maken voor Excel of verdere analyse

## Inhoud
- `wkof_donateursdata.ipynb` - [Notebook fondsenwervingsdata met verwerking en verkennende analyse](https://github.com/zch4/fondsenwerving_data_kaggle/blob/main/wkof_donateursdata.ipynb)
- `data/bron/` - bron-CSV's voor de notebook
- `requirements.txt` - minimale Python-afhankelijkheden

## Dataset
Bron: [Mock Nonprofit Fundraising Data op Kaggle](https://www.kaggle.com/datasets/grantstancliff/mock-nonprofit-fundraising-data)
